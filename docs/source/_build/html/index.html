DJI Robomaster and Tello Python API¶

A set of python programming interfaces for controlling DJI Robomaster EP and DJI Tello EDU in SDK mode via python codes. This set of inferfaces handles establishing SDK connection to DJI Robomaster EP and DJI Tello EDU and controls those robots using function calls.

Important!¶

Before using the codes, download h264 and opus libraries from https://github.com/dji-sdk/RoboMaster-SDK/tree/master/sample_code/RoboMasterEP/stream/decoder and follow the instructions in the link to compile.
